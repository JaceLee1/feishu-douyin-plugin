# ⚠️ 请手动启用 GitHub Pages

## 步骤（2 分钟完成）

### 1. 打开仓库设置

点击或复制访问：
```
https://github.com/JaceLee1/feishu-douyin-plugin/settings/pages
```

### 2. 配置 Pages

在 **Build and deployment** 部分：

1. **Source**: 选择 `Deploy from a branch`
2. **Branch**: 选择 `main`
3. **Folder**: 选择 `/ (root)`
4. 点击 **Save**

### 3. 等待部署

等待 1-2 分钟，页面会自动刷新显示访问地址

### 4. 访问地址

部署完成后，访问：
```
https://JaceLee1.github.io/feishu-douyin-plugin/index.html
```
