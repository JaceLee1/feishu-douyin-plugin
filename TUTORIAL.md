## 1. GitHub 仓库创建

### 1.1 访问 GitHub
打开浏览器，访问：https://github.com

### 1.2 创建新仓库
1. 点击右上角 **「+」** 图标
2. 选择 **「New repository」**

### 1.3 填写仓库信息
```
Repository name: feishu-douyin-plugin
Description: 飞书多维表格插件 - 抖音数据提取
Visibility:
  ● Public (公开，推荐)
  ○ Private (私有)
```
- **不要** 勾选 "Add a README file"
- **不要** 添加 .gitignore
- **不要** 选择 license

### 1.4 创建仓库
点击绿色按钮 **「Create repository」**

## 2. 代码推送

### 方法一：使用部署脚本（推荐）

1. 双击运行项目中的 `deploy.bat` 文件
2. 输入你的 GitHub 用户名
3. 按提示完成推送

### 方法二：手动命令

打开 Git Bash，执行：

```bash
# 进入项目目录
cd C:/Users/asus/feishu-douyin-plugin

# 添加远程仓库（替换 YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/feishu-douyin-plugin.git

# 切换分支
git branch -M main

# 推送代码
git push -u origin main
```

### 认证问题解决方法

如果推送时提示认证失败：

1. 访问：https://github.com/settings/tokens
2. 点击 **「Generate new token」** → **「Create new token (classic)」**
3. 填写备注（如：feishu-plugin-deploy）
4. 勾选权限：**repo** (全选)
5. 点击底部 **「Generate token」**
6. **复制 Token**（只显示一次，请妥善保存）
7. 推送时使用 Token 代替密码

## 3. GitHub Pages 配置

### 3.1 进入 Settings
1. 打开你的 GitHub 仓库页面
2. 点击顶部导航栏的 **「Settings」**

### 3.2 配置 Pages
1. 点击左侧菜单的 **「Pages」**
2. 在 **Build and deployment** 部分：
   - **Source**: Deploy from a branch
   - **Branch**: 选择 `main`
   - **Folder**: 选择 `/ (root)`
3. 点击 **「Save」**

### 3.3 等待部署
- 等待 1-2 分钟
- 刷新页面，看到顶部绿色提示表示部署成功

### 3.4 获取访问地址
```
https://YOUR_USERNAME.github.io/feishu-douyin-plugin/index.html
```

**重要**：复制此 URL，后面飞书配置需要用到！

## 4. 飞书应用创建

### 4.1 访问飞书开放平台
打开浏览器，访问：https://open.feishu.cn/

### 4.2 登录
使用飞书 App 扫码登录或账号密码登录

### 4.3 创建应用
1. 点击顶部导航栏 **「企业自建」**
2. 点击 **「创建应用」**

### 4.4 填写应用信息
```
应用名称：抖音数据提取
应用图标：点击选择一个音乐相关的图标
应用描述：输入抖音视频链接，自动提取数据到表格
```
3. 点击 **「创建」**

## 5. 插件配置

### 5.1 添加多维表格插件

1. 在应用管理页面，点击左侧 **「应用功能」**
2. 点击 **「添加功能」** 按钮
3. 在弹出的窗口中选择 **「多维表格插件」**
4. 填写插件信息：

```
插件名称：抖音数据提取
插件描述：输入抖音链接自动提取视频数据
前端页面 URL：[填入 GitHub Pages 地址]
              https://YOUR_USERNAME.github.io/feishu-douyin-plugin/index.html
```

5. 点击 **「确定」**

### 5.2 配置权限

1. 点击左侧菜单 **「权限管理」**
2. 点击 **「申请权限」** 按钮
3. 搜索框输入 "bitable"
4. 勾选以下权限：
   - `bitable:table` - 多维表格
   - `bitable:table:read` - 读取表格数据
   - `bitable:table:write` - 写入表格数据
5. 点击 **「确定」**

### 5.3 发布应用

1. 点击左侧菜单 **「版本管理与发布」**
2. 点击 **「创建版本」**
3. 填写版本信息：
   ```
   版本号：1.0.0
   更新说明：初始版本发布
   ```
4. 点击 **「提交审核」**
5. 企业自建应用通常自动通过审核
6. 审核通过后，点击 **「发布」**

## 6. 表格字段创建

### 6.1 创建多维表格

1. 打开飞书桌面客户端
2. 点击左侧 **「云文档」**
3. 点击 **「+」** → **「多维表格」**
4. 命名表格，如「抖音数据管理」

### 6.2 创建字段

依次创建以下 9 个字段：

| 序号 | 字段名称 | 字段类型 | 操作步骤 |
|-----|---------|---------|---------|
| 1 | 视频链接 | 文本 | 默认已有，重命名即可 |
| 2 | 视频文案 | 多行文本 | + 添加字段 → 多行文本 |
| 3 | 封面图 | 附件 | + 添加字段 → 附件 |
| 4 | 点赞数 | 数字 | + 添加字段 → 数字 |
| 5 | 评论数 | 数字 | + 添加字段 → 数字 |
| 6 | 转发数 | 数字 | + 添加字段 → 数字 |
| 7 | 播放数 | 数字 | + 添加字段 → 数字 |
| 8 | 发布时间 | 日期 | + 添加字段 → 日期 |
| 9 | 作者昵称 | 文本 | + 添加字段 → 文本 |

### 字段创建详细步骤

1. 点击表头右侧的 **「+」** 号
2. 选择字段类型
3. 输入字段名称
4. 点击 **「确定」**
5. 重复以上步骤创建所有字段

## 7. 使用测试

### 7.1 添加插件

1. 在多维表格页面，点击右上角 **「···」**
2. 选择 **「插件」**
3. 找到 **「抖音数据提取」**
4. 点击 **「添加」**

### 7.2 使用插件

1. 点击右侧边栏的 **插件图标**（拼图形状）
2. 选择 **「抖音数据提取」**
3. 在插件界面：
   - 粘贴抖音视频链接
   - 点击 **「提取数据」**
4. 等待几秒后，数据将自动写入表格

### 7.3 测试链接

可以使用以下格式的抖音链接测试：
- `https://v.douyin.com/xxxxx/`
- `https://www.douyin.com/video/xxxxxx`

## 常见问题排查

### Q1: 飞书插件无法加载？

**原因 1**：GitHub Pages 未部署成功
- 检查 GitHub Pages 设置是否正确
- 等待 1-2 分钟让部署完成

**原因 2**：URL 填写错误
- 确保是 HTTPS 开头
- 确保路径包含 index.html

**原因 3**：跨域问题
- GitHub Pages 地址必须可公开访问

### Q2: 推送代码失败？

**错误**：Authentication failed
- 使用 Personal Access Token
- 访问 https://github.com/settings/tokens 生成

**错误**：Repository not found
- 确保已在 GitHub 创建仓库
- 检查用户名和仓库名是否正确

### Q3: 数据提取失败？

**原因 1**：链接格式错误
- 确保是抖音官方链接格式

**原因 2**：API 服务限制
- 免费 API 可能有调用限制
- 稍后重试或更换 API 服务

**原因 3**：字段不匹配
- 检查字段名称是否完全一致
- 检查是否所有必填字段都已创建

### Q4: 封面图无法显示？

- 确保「封面图」字段类型是「附件」或「图片」
- 检查图片 URL 是否可公开访问
