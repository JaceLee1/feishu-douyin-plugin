# 飞书插件 - 5 分钟快速配置指南

## 前提
- 你有飞书账号且是企业管理员（或能创建企业自建应用）

---

## 步骤（跟着做，5 分钟完成）

### 1. 访问飞书开放平台

打开：**https://open.feishu.cn/**

用飞书 App 扫码登录

---

### 2. 创建应用

1. 点击顶部 **「创建应用」**
2. 选择 **「企业自建应用」**
3. 填写：
   ```
   应用名称：抖音数据提取
   ```
   （其他随便填）
4. 点击 **「创建」**

---

### 3. 添加多维表格插件

1. 进入应用后，左侧菜单找到 **「应用功能」**
2. 点击 **「添加功能」**
3. 选择 **「多维表格插件」**
4. 填写：
   ```
   插件名称：抖音数据提取
   前端页面 URL: https://JaceLee1.github.io/feishu-douyin-plugin/index.html
   ```
5. 点击 **「确定」**

---

### 4. 申请权限

1. 左侧菜单 **「权限管理」**
2. 点击 **「申请权限」**
3. 搜索框输入 `bitable`
4. 勾选：
   - [x] bitable:table
   - [x] bitable:table:read
   - [x] bitable:table:write
5. 点击 **「确定」**

---

### 5. 添加试用成员（关键！）

1. 左侧菜单 **「版本管理与发布」**
2. 点击 **「创建版本」**
3. 填写：
   ```
   版本号：1.0.0
   更新说明：测试版本
   ```
4. 点击 **「提交审核」**
5. 提交后，在版本列表找到刚创建的版本
6. 点击 **「添加试用成员」**
7. 选择 **你自己**
8. 点击 **「保存」**

---

### 6. 在飞书多维表格中使用

1. 打开飞书桌面客户端
2. 新建或打开一个多维表格
3. 点击右侧边栏的 **「+」** 或 **插件图标**
4. 找到 **「抖音数据提取」**
5. 点击打开

---

## 如果看不到插件

1. 确认你已添加为试用成员
2. 退出飞书重新登录
3. 等 1-2 分钟再试

---

## 创建表格字段

使用前，先在多维表格创建以下字段：

| 字段名 | 类型 |
|-------|------|
| 视频链接 | 文本 |
| 视频文案 | 多行文本 |
| 封面图 | 附件 |
| 点赞数 | 数字 |
| 评论数 | 数字 |
| 转发数 | 数字 |
| 播放数 | 数字 |
| 发布时间 | 日期 |
| 作者昵称 | 文本 |

---

## 常见问题

**Q: 没有「企业自建」选项？**
A: 你需要是企业管理员，或者让管理员帮你创建

**Q: 申请权限被拒绝？**
A: 需要企业管理员审批

**Q: 插件加载后是空白？**
A: 检查 GitHub Pages 地址是否正确
