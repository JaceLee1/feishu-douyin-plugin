# 飞书多维表格插件 - 抖音数据提取

## 📦 部署步骤（从 0 到 1）

### 第一步：创建飞书应用

1. 登录 [飞书开放平台](https://open.feishu.cn/)
2. 点击左上角 **「创建应用」**
3. 选择 **「创建企业自建应用」**
4. 填写应用信息：
   - 应用名称：抖音数据提取
   - 应用图标：可选一个音乐相关的图标
   - 应用描述：提取抖音视频数据到多维表格
5. 点击 **「创建」**

### 第二步：配置应用权限

1. 在应用管理页面，点击左侧 **「权限管理」**
2. 点击 **「申请权限」**
3. 搜索并添加以下权限：
   - `bitable:table` - 多维表格读取权限
   - `bitable:table:read` - 读取表格数据
   - `bitable:table:write` - 写入表格数据
4. 点击 **「确定」** 提交申请

### 第三步：创建多维表格插件

1. 在应用管理页面，点击左侧 **「应用功能」**
2. 点击 **「添加功能」** → 选择 **「多维表格插件」**
3. 填写插件信息：
   - 插件名称：抖音数据提取
   - 插件描述：输入抖音链接自动提取视频数据
   - 服务端 URL：本插件为纯前端实现，此项可留空或填占位 URL
   - **前端页面 URL**：填写你的静态资源地址（见第四步）

### 第四步：部署前端代码

#### 方式一：使用飞书云空间（推荐）

1. 将 `index.html` 和 `main.js` 上传到任意静态托管服务：
   - GitHub Pages
   - Vercel
   - Netlify
   - 或你自己的服务器

2. 获取公开访问 URL，例如：
   ```
   https://your-domain.com/feishu-douyin-plugin/index.html
   ```

3. 将此 URL 填入插件配置的 **「前端页面 URL」**

#### 方式二：本地测试

1. 使用本地服务器运行：
   ```bash
   # 使用 Python
   cd feishu-douyin-plugin
   python -m http.server 8080

   # 或使用 Node.js
   npx serve feishu-douyin-plugin
   ```

2. 使用内网穿透工具（如 ngrok）暴露本地服务：
   ```bash
   ngrok http 8080
   ```

3. 将生成的 https URL 填入插件配置

### 第五步：配置发布

1. 返回应用管理页面
2. 点击左侧 **「版本管理与发布」**
3. 点击 **「创建版本」**
4. 填写版本信息和更新说明
5. 提交审核（企业自建应用通常自动通过）
6. 审核通过后，点击 **「发布」**

### 第六步：安装应用到多维表格

1. 打开任意飞书多维表格
2. 点击右上角 **「···」** → **「插件」**
3. 找到 **「抖音数据提取」** 插件
4. 点击 **「添加」** 安装

## 📋 多维表格字段创建说明

在使用插件前，请先创建好以下字段：

| 字段名称 | 字段类型 | 说明 |
|---------|---------|------|
| 视频链接 | 文本 | 抖音分享链接 |
| 视频文案 | 多行文本 | 视频描述/标题 |
| 封面图 | 附件/图片 | 视频封面 |
| 点赞数 | 数字 | 点赞数量 |
| 评论数 | 数字 | 评论数量 |
| 转发数 | 数字 | 转发/分享数量 |
| 播放数 | 数字 | 播放量 |
| 发布时间 | 日期 | 视频发布时间 |
| 作者昵称 | 文本 | 视频作者 |

### 创建字段步骤：

1. 打开飞书多维表格
2. 点击表头右侧的 **「+」** 添加字段
3. 按照上表依次创建 9 个字段
4. 字段名称必须完全一致（插件按名称匹配）

## 📖 使用说明

### 基本使用流程

1. **打开插件**
   - 在飞书多维表格中
   - 点击右侧边栏的插件图标
   - 选择「抖音数据提取」

2. **输入链接**
   - 复制抖音视频分享链接
   - 粘贴到插件输入框

3. **提取数据**
   - 点击「提取数据」按钮
   - 等待解析完成
   - 数据自动写入表格

### 支持的链接格式

- ✅ 抖音分享链接：`https://v.douyin.com/xxxxx/`
- ✅ 抖音完整链接：`https://www.douyin.com/video/xxxxxx`
- ✅ 去水印链接：`https://iesdouyin.com/share/video/xxxxxx`

### 数据更新逻辑

- 如果表格中已存在相同链接的记录，会自动**更新**该记录
- 如果是新链接，会**新增**一条记录
- 去重判断依据：视频链接字段

### 常见问题

**Q: 提示「没有权限写入表格」？**
A: 请确保：
- 已给插件授予多维表格写入权限
- 当前用户对表格有编辑权限

**Q: 提示「表格字段不匹配」？**
A: 请检查是否已创建所有必需字段，字段名称需完全一致

**Q: 解析失败？**
A: 可能原因：
- 链接格式不正确
- 视频已被删除或设为私密
- API 服务暂时不可用，请稍后重试

**Q: 封面图无法显示？**
A: 确保「封面图」字段类型设置为「附件」或「图片」

## 🔧 自定义配置

### 修改解析 API

如果需要更换解析服务，编辑 `main.js` 中的 API 配置：

```javascript
// 方法 1: 使用 TikHub（需注册获取 API Key）
const DOUYIN_PARSER_API = 'https://api.tikhub.io/api/v1/short_video/';

// 方法 2: 使用其他服务
const BACKUP_PARSER_APIS = [
  'https://your-api.com/parse?url='
];
```

### 推荐解析服务

| 服务商 | 地址 | 特点 |
|-------|------|------|
| TikHub | https://tikhub.io/ | 稳定，有免费额度 |
| 轻解析 | https://api.qjapi.com/ | 免费，无需 Key |
| TENAPI | https://tenapi.cn/ | 免费备用 |

## ⚠️ 注意事项

1. **API 限制**：免费解析 API 可能有调用频率限制，大量使用建议购买付费服务
2. **链接时效**：抖音分享链接有时效性，请尽快提取
3. **隐私视频**：私密/已删除的视频无法获取数据
4. **合规使用**：请遵守抖音和飞书的使用条款
