# 飞书多维表格 - 简单使用方案

## 前提条件
- 你有飞书账号
- 能访问飞书多维表格

---

## 方案一：网页嵌入（推荐，5 分钟搞定）

### 步骤

#### 1. 打开飞书多维表格
- 打开飞书
- 新建一个多维表格

#### 2. 创建字段
按顺序创建以下 9 个字段：

| 字段名 | 类型 |
|-------|------|
| 视频链接 | 文本 |
| 视频文案 | 多行文本 |
| 封面图 | 附件/图片 |
| 点赞数 | 数字 |
| 评论数 | 数字 |
| 转发数 | 数字 |
| 播放数 | 数字 |
| 发布时间 | 日期 |
| 作者昵称 | 文本 |

#### 3. 添加网页嵌入
1. 在多维表格右侧，点击 **「+」** 添加扩展面板
2. 选择 **「网页嵌入」** 或 **「添加链接」**
3. 填入 URL：
   ```
   https://JaceLee1.github.io/feishu-douyin-plugin/index.html
   ```
4. 命名：`抖音数据提取`

#### 4. 使用
- 在右侧面板打开插件
- 粘贴抖音链接
- 点击提取
- 数据会显示预览，然后手动复制到表格

## 方案二：本地运行（无需部署）

### 直接在浏览器打开使用

1. 双击打开项目中的 `index.html` 文件
2. 或者在命令行执行：
   ```bash
   cd C:/Users/asus/feishu-douyin-plugin
   npx serve .
   ```
3. 浏览器访问显示的地址

### 使用流程
1. 打开插件页面
2. 粘贴抖音链接，点击提取
3. 查看提取的数据预览
4. 手动复制到飞书表格

## 方案三：飞书自建应用（无需发布）

### 创建企业自建应用

1. 访问：https://open.feishu.cn/
2. 登录飞书
3. 点击 **「企业自建」** → **「创建应用」**
4. 填写：
   - 应用名称：抖音数据提取
   - 其他随意填
5. 点击 **「创建」**

### 添加插件

1. 左侧 **「应用功能」** → **「添加功能」**
2. 选择 **「多维表格插件」**
3. 填写：
   - 插件名称：抖音数据提取
   - **前端页面 URL**: `https://JaceLee1.github.io/feishu-douyin-plugin/index.html`
4. 点击 **「确定」**

### 添加权限（简化版）

1. 左侧 **「权限管理」**
2. 点击 **「申请权限」**
3. 搜索 `bitable`，勾选所有相关权限
4. 点击 **「确定」**

### 直接使用（不发布）

1. 左侧 **「版本管理与发布」**
2. 点击 **「创建版本」** → 随便填 → **「提交」**
3. **不用等审核**，在应用版本页面点击 **「试用」**
4. 添加你自己为试用成员
5. 保存

### 在飞书中打开

1. 打开飞书桌面客户端
2. 进入任意多维表格
3. 右侧边栏 → 插件 → 找到你的应用
4. 打开使用

优点：
- ✅ 无需审核
